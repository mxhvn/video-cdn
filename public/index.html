MP4 uploader qua GitHub Contents API

Flow: chọn file MP4 → Pages Function nhận file → commit vào repo tại public/uploads/ → Cloudflare Pages tự deploy lại → trả về URL Pages và jsDelivr.

Dùng tốt cho demo/MVP nhỏ. Không nên dùng cho video lớn hoặc upload thường xuyên. GitHub web UI có giới hạn 25 MiB; GitHub Contents API hỗ trợ tối đa 100 MB theo tài liệu, nhưng thực tế nên giữ nhỏ hơn đáng kể để giảm lỗi và thời gian deploy.
